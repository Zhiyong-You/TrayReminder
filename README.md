# TrayReminder

Windows タスクトレイに常駐する、シンプルなリマインダーアプリです。

## 動作環境

| 項目 | 要件 |
|---|---|
| OS | Windows 10 / 11 |
| ランタイム | .NET 8 (Windows) |

---

## インストールと起動

1. リポジトリをクローン、またはソースを取得します
2. 以下のコマンドでビルドします

```
dotnet build
```

3. 生成された `bin\Debug\net8.0-windows\TrayReminder.exe` を実行します
4. タスクトレイにアイコンが表示されます

---

## 基本的な使い方

### アプリの起動・終了

- **起動**: `TrayReminder.exe` を実行すると、タスクトレイにアイコンが表示されます
- **主画面を開く**: トレイアイコンをダブルクリック、または右クリック→「主画面を開く」
- **終了**: トレイアイコンを右クリック→「終了」
- **主画面を閉じる**: ウィンドウの × ボタンで閉じても、アプリはトレイに残り続けます（終了ではありません）

---

## リマインダーの管理

### 一覧画面

主画面にリマインダーの一覧が表示されます。

| 列 | 内容 |
|---|---|
| タイトル | リマインダーの名前 |
| 通知日時 | 次回通知される日時 |
| 有効 | チェックが入っていると通知対象になります |

### 追加

リマインダーを追加する方法は2つあります。

**方法1: 主画面から追加**
1. 主画面下部の「追加」ボタンをクリック
2. 各項目を入力して「OK」をクリック

**方法2: トレイから素早く追加（QuickAdd）**
1. トレイアイコンを右クリック→「新規リマインダー」
2. タイトル・日付・時刻を入力して「保存」をクリック

### 編集

1. 一覧からリマインダーを選択
2. 「編集」ボタンをクリック
3. 内容を変更して「OK」をクリック

### 削除

1. 一覧からリマインダーを選択
2. 「削除」ボタンをクリック

---

## 入力項目

| 項目 | 内容 |
|---|---|
| タイトル | リマインダーの名前（必須） |
| メモ | 補足情報（省略可） |
| 日付 | DatePicker で選択 |
| 時刻 | 00:00 〜 23:55 を 5 分刻みで選択 |
| 繰り返し | 通知の繰り返しパターン（後述） |
| 有効 | チェックを外すと通知が発生しません |

---

## 繰り返し設定

| 設定値 | 動作 |
|---|---|
| None | 1回だけ通知。完了後は終了 |
| Daily | 毎日、同じ時刻に通知 |
| Weekly | 毎週、同じ曜日・同じ時刻に通知 |
| Workday | 平日（月〜金）のみ、同じ時刻に通知。金・土・日の次は月曜 |

繰り返しリマインダーを「完了」すると、自動的に次回の通知日時が設定され、再び通知対象になります。

---

## 通知ポップアップ

指定した日時になると、画面右下付近にポップアップが表示されます。

| ボタン | 動作 |
|---|---|
| 完了 | リマインダーを完了にします。繰り返し設定がある場合は次回へ進みます |
| 5分後 | 5分後に再通知します（スヌーズ） |
| 10分後 | 10分後に再通知します（スヌーズ） |
| 閉じる | ポップアップを閉じます。約5分後に再通知されます |
| × ボタン | 「閉じる」と同じ動作です |

---

## スヌーズについて

- 「5分後」「10分後」を選ぶと、`SnoozeUntil`（再通知時刻）が設定されます
- アプリを再起動してもスヌーズ状態は保持されます
- スヌーズ時刻を過ぎると、自動的に再通知されます
- 「完了」するとスヌーズ設定はクリアされます

---

## データの保存場所

リマインダーのデータは以下の場所に JSON 形式で保存されます。

```
%LOCALAPPDATA%\TrayReminder\reminders.json
```

例: `C:\Users\<ユーザー名>\AppData\Local\TrayReminder\reminders.json`

アプリを再インストールしてもこのファイルが残っていれば、データは引き継がれます。
