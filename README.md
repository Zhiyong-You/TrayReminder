# TrayReminder

Windows タスクトレイに常駐する、シンプルなリマインダーアプリです。

## 動作環境

| 項目 | 要件 |
|---|---|
| OS | Windows 10 / 11 |
| ランタイム | .NET 8 (Windows) |

---

## インストールと起動

1. リポジトリをクローン、またはソースを取得します
2. 以下のコマンドでビルドします

```
dotnet build
```

3. 生成された `bin\Debug\net8.0-windows\TrayReminder.exe` を実行します
4. タスクトレイにアイコンが表示されます

---

## 基本的な使い方

### アプリの起動・終了

- **起動**: `TrayReminder.exe` を実行すると、タスクトレイにアイコンが表示されます
- **主画面を開く**: トレイアイコンをダブルクリック、または右クリック→「主画面を開く」
- **終了**: トレイアイコンを右クリック→「終了」
- **主画面を閉じる**: ウィンドウの × ボタンで閉じても、アプリはトレイに残り続けます（終了ではありません）

---

## リマインダーの管理

### 一覧画面

主画面にリマインダーの一覧が表示されます。一覧は以下の順で自動ソートされます。

1. 未完了が上、完了済みが下
2. 有効が上、無効が下
3. 通知日時が早い順
4. タイトルのアルファベット順

#### 視覚的なステータス表示

| 表示 | 意味 |
|---|---|
| 通常（白文字） | 有効・未完了・未来日時 |
| グレー文字 | 無効（通知が発生しません） |
| 打ち消し線・グレー | 完了済み |

### フィルタ

一覧上部のドロップダウンで表示するアイテムを絞り込めます。

| フィルタ | 表示条件 |
|---|---|
| すべて | 全件表示 |
| 有効のみ | `IsEnabled = true` のアイテム |
| 未完了のみ | `IsCompleted = false` のアイテム |
| 期限超過のみ | 有効・未完了・通知日時が現在より過去のアイテム |

### 詳細エリアのステータスバッジ

一覧でアイテムを選択すると右側に詳細が表示されます。状態に応じてバッジが表示されます。

| バッジ | 状態 |
|---|---|
| ✓ 完了（紫） | 完了済み |
| ○ 無効（グレー） | 無効に設定されている |
| ⚠ 期限超過（オレンジ） | 有効・未完了だが通知日時が過去 |
| バッジなし | 正常（有効・未完了・未来日時） |

### 追加

リマインダーを追加する方法は2つあります。

**方法1: 主画面から追加**
1. 主画面下部の「追加」ボタンをクリック
2. 各項目を入力して「OK」をクリック

**方法2: トレイから素早く追加（QuickAdd）**
1. トレイアイコンを右クリック→「新規リマインダー」
2. タイトル・日付・時刻を入力して「保存」をクリック

### 編集

1. 一覧からリマインダーを選択
2. 「編集」ボタンをクリック
3. 内容を変更して「OK」をクリック

### 削除

**1件削除**
1. 一覧からリマインダーを選択
2. 「削除」ボタンをクリック
3. 確認ダイアログで「削除」をクリック

**完了済みを一括削除**
1. 主画面下部の「完了済みを削除」ボタンをクリック
2. 確認ダイアログで件数を確認して「削除」をクリック

### スヌーズ（主画面から）

1. 一覧からリマインダーを選択
2. 「スヌーズ」ボタンをクリック（10分後に再通知）

## 入力項目

| 項目 | 内容 |
|---|---|
| タイトル | リマインダーの名前（必須） |
| メモ | 補足情報（省略可） |
| 日付 | DatePicker で選択 |
| 時刻 | 00:00 〜 23:55 を 5 分刻みで選択 |
| 繰り返し | 通知の繰り返しパターン（後述） |
| 有効 | チェックを外すと通知が発生しません |

---

## 繰り返し設定

| 設定値 | 動作 |
|---|---|
| None | 1回だけ通知。完了後は終了 |
| Daily | 毎日、同じ時刻に通知 |
| Weekly | 毎週、同じ曜日・同じ時刻に通知 |
| Workday | 平日（月〜金）のみ、同じ時刻に通知。金・土・日の次は月曜 |
| Monthly | 毎月、同じ日・同じ時刻に通知 |
| Yearly | 毎年、同じ月日・同じ時刻に通知 |

繰り返しリマインダーを「完了」すると、現在日時より未来になるまで自動的に次回の通知日時が計算され、再び通知対象になります。

#### 月末・閏年の扱い

| ケース | 動作 |
|---|---|
| Monthly で翌月に同じ日がない（例：1/31→2月） | その月の最終日を使用（2/28 または 2/29） |
| Yearly で翌年に 2/29 がない（うるう年→非うるう年） | 2/28 を使用 |

## 通知ポップアップ

指定した日時になると、画面**右下**（タスクバーの上）にポップアップが表示されます。

- 複数の通知が同時に発生した場合は、下から順に積み上げて表示されます
- ポップアップはドラッグで移動できます（ボタン以外の領域をドラッグ）
- 操作しない場合は 30 秒後に自動で閉じます（5分後に再通知）
- アイテムを削除すると、表示中の通知ポップアップも自動で閉じます

| ボタン | 動作 |
|---|---|
| 完了 | リマインダーを完了にします。繰り返し設定がある場合は次回へ進みます |
| 5分後 | 5分後に再通知します（スヌーズ） |
| 10分後 | 10分後に再通知します（スヌーズ） |
| 閉じる | ポップアップを閉じます。約5分後に再通知されます |
| × ボタン | 「閉じる」と同じ動作です |

---

## スヌーズについて

- 「5分後」「10分後」を選ぶと、`SnoozeUntil`（再通知時刻）が設定されます
- アプリを再起動してもスヌーズ状態は保持されます
- スヌーズ時刻を過ぎると、自動的に再通知されます
- 「完了」するとスヌーズ設定はクリアされます

## 自動起動（ログオン時の自動起動）

### ON にする方法

1. タスクトレイのアイコンを右クリック
2. 「ログオン時に自動起動」をクリック（チェックが付きます）

次回 Windows ログオン時から自動的に起動します。自動起動時はウィンドウを表示せずトレイ常駐のみで起動します。主画面を開くにはトレイアイコンをダブルクリックしてください。

### OFF にする方法

1. タスクトレイのアイコンを右クリック
2. 「ログオン時に自動起動」をクリック（チェックが外れます）

### 仕組み

`HKCU\Software\Microsoft\Windows\CurrentVersion\Run` レジストリキーに以下の値を登録します。管理者権限は不要です。

```
名前: TrayReminder
値:   "C:\...\TrayReminder.exe" --startup
```

### 注意事項

| ケース | 対処 |
|---|---|
| exe を別の場所に移動した | トレイメニューで一度 OFF にしてから再度 ON にする（パスが更新されます） |
| アンインストール時 | 事前にトレイメニューで OFF にしてからアンインストールしてください |

## データの保存場所

リマインダーのデータは以下の場所に JSON 形式で保存されます。

```
%LOCALAPPDATA%\TrayReminder\reminders.json
```

例: `C:\Users\<ユーザー名>\AppData\Local\TrayReminder\reminders.json`

アプリを再インストールしてもこのファイルが残っていれば、データは引き継がれます。
